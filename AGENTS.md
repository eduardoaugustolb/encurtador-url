# Package Manager

This project uses **bun** as the package manager. Do NOT use npm or pnpm.
- Install dependencies: `bun install`
- Add packages: `bun add <package>`
- Dev dependencies: `bun add -d <package>`
- Run scripts: `bun run <script>`
